# 🐍 Python Projects

This repository contains two Python projects developed to practice Python programming and file handling concepts.

## 📌 Projects

### 🎓 Student Result Manager

A command-line Python application to manage student records and calculate results.

**Features**
- Add student records
- Update student information
- Search students
- Display all student results
- Calculate grades

### 🔐 Password Manager

A Python application that securely stores and manages passwords.

**Features**
- Save passwords
- View saved passwords
- Search passwords
- Delete passwords

## 🛠️ Technologies Used

- Python 3
- File Handling
- Functions
- Loops
- Conditional Statements

## 🚀 Getting Started

Clone this repository:

```bash
git clone https://github.com/ilma2611/PYTHON.git
```

Open the project folder:

```bash
cd PYTHON
```

Run the desired Python file:

```bash
python filename.py
```

> Replace `filename.py` with your actual Python file name.
